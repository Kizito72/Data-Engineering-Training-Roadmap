# Data Engineering Training Roadmap

A 3-month roadmap and execution action plan for becoming job-ready as a Data Engineer / Data Analyst.

## 🚀 Features
- **Interactive roadmap** — 3 phases across 12 weeks
- **Study timer** — with beep on start, 10-minute alarm, and end alarm
- **Checkboxes** — track every task and phase
- **Daily log** — record each session
- **Schedule viewer** — your fixed weekly blocks
- **Google Calendar integration** — Apps Script code to auto-create all study events with 30-min phone reminders
- **Customizable study times** — update your schedule anytime
- **Progress bars** — overall + per-phase tracking

## 📅 Roadmap Summary
| Phase | Weeks | Focus |
|-------|-------|-------|
| Phase 1 | 1–4 | SQL + Python Foundations |
| Phase 2 | 5–8 | ETL Pipelines + Data Modeling |
| Phase 3 | 9–12 | Google Cloud + Final Portfolio Project |

## 🔔 Google Calendar Setup
1. Go to [script.google.com](https://script.google.com)
2. New project → paste code from the **Google Calendar** tab
3. Run `setupStudySessions` → authorize → done!

## 📦 Deploy to GitHub Pages
1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site is live at `https://[username].github.io/[repo-name]`
